Detailed Description

This is class used to read GameObject from XML file. This use external gameObjectPlant and XML Reader Class don't allocate any memory and don't destroy gameObjectPlant and XMLReader. gameObjectPlant and XML reader must be given when class is created, and must be free by caller; hasNext method check if there is any prefab to read in indicated file getNext method return read prefab, but you must down cast returned prefab.
