Detailed Description

template<class gObject>
class GameCollectionIterator< gObject >

Game iterator, used to move forward the game collection. It offer two main method hasNext checking if there are next object in collection, and getNext which move to next object, and return it. Iterator also stores the pointer to previous list element so it can be used to remove object from collection without looking for object. You can use iterator like STL iterator e.g. for(it = col.getIterator(); it.hasNext(); it++) { fun(*it); it->doSth;} or in old way while(it.hasNext()) {obj = it.getNext();)
