Detailed Description

Collider component stores data about collider which game object has. To provide good accuracy, collider can be build from number of small spheres, but to provide collider's high efficiency, it is required to set inaccurate collider which should cover all small collider. Small collider will be checked only when there is inaccurate collision. If there is less then two small colliders, only inaccurate collisions will be checked.
